# docs/ 폴더

이 폴더에는 RAG 기능이 검색할 내부 문서가 포함됩니다.

## 지원 파일 형식

- `.txt`: 일반 텍스트 파일
- `.md`: Markdown 파일
- `.pdf`: PDF 문서

## 문서 추가 방법

1. 이 폴더에 문서 파일을 추가합니다
2. Agent를 재시작하면 자동으로 벡터화되어 검색 가능해집니다
3. 첫 실행 시 임베딩 생성 (10-30초 소요)
4. 이후 실행은 캐시 사용 (0.5초 이내)

## 예제 파일

- `test_doc.txt`: LENA 플랫폼 관련 정보
- `project_x.txt`: Project X 프로젝트 정보

## 주의사항

- 민감한 정보는 저장소에 커밋하지 마세요!
- `docs/` 폴더를 `.gitignore`에추가하는 것을 권장합니다 (배포 시)
- 벡터 DB 캐시는 `chroma_db/` 폴더에 저장됩니다
